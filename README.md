# literature
如果你对唐诗宋词感兴趣，想搜索关于描述春天的唐诗宋词，那么源码应用可以帮助你！源码利用搜索引擎，可以根据所要搜索的诗词关键字，找到相关的唐诗宋词，包括内容，作者，标题。数据库收集了500多首唐诗宋词，来源于互联网，不敢保证100%数据正确性!
